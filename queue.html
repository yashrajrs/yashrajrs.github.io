Queue

  • Queue is FIFO (first in first out) data structure.
  • It provides three basic methods enqueue, dequeue and peek.
  • Enqueue adds the element to the end of the queue.
  • Dequeue the first element from the queue, if the queue is not empty.
  • Peek returns the value of first element of the queue, if there exists one.

public class Queue {
    /** public class Node{
     * Node next;
     * Object data;	
     */
    public Node first, last;

    public void enqueue(Object data) {
        if (first == null) {
            last = new Node(data);
            first = last;
        } else {
            last.setNext(new Node(data));
            last = last.getNext();
        }
    }

    public Object dequeue() {
        if (first != null) {
            Object data = first.getData();
            first = first.getNext();
            return data;
        }
        return null;
    }

    public Object peek() {
        if (first==null){
            return null;
        }
        return first.getData();
    }
}
