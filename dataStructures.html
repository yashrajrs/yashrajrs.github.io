Stack

  • Stack is LIFO (last in first out) data structure.
  • It provides four basic methods push, pop, peek and isEmpty.
  • Push pushes the element to the top of the stack.
  • Pop removes the top most element from the stack, if the stack is not empty.
  • Peek returns the value of top most element of the stack, if there exists one.
  • IsEmpty return true when stack is empty, false otherwise.
  • Example

Queue

  • Queue is FIFO (first in first out) data structure.
  • It provides three basic methods enqueue, dequeue and peek.
  • Enqueue adds the element to the end of the queue.
  • Dequeue the first element from the queue, if the queue is not empty.
  • Peek returns the value of first element of the queue, if there exists one.
  • Example

LinkedList

  • Linked list creates list of elements using next pointers.
  • It usually provides three methods getFirst, appendToTail and delete.
  • GetFirst returns the first element from the list, if there exists one.
  • AppendToTail appends an element to the tail of the list.
  • Delete deletes the element from the list, if it exits.
  • Example
  • Some more variations of linked list are DoublyLinkedList and CircularLinkedList

Binary Search Tree

  • Binary Search Tree creates a binary tree such that an element has two children, left and right.
  • The parent's value is greater than left child and parent's value is less than right child.
  • It usually provides with addData, delete, printInOrder, printPreOrder and printPostOrder methods.
  • Example
